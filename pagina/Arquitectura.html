Pila de Tecnologías


React Native (FrontEnd) Con el fin de facilitar el diseño de las interfaces en la solución, se implementará React Native que permite desarrollar el FrontEnd basado en componentes reutilizables. Además pensando en la experiencia de usuario React optimiza el desempeño de la página en el cliente y facilita ser renderizado en dispositivos móviles sin importar SO.

Node.js (Servidor de aplicaciones) Es un entorno de ejecución que permite desarrollar sistemas escalables eficientemente, ya que está orientado a eventos asíncronos que falicitan el desarrollo sin preocuparse por las funciones bloqueantes.

MySQL (Base de datos) Se implementará una base de datos relacional con el objetivo de que el diseño sea mas estable usando una metodologia clasica (Entidad - Relación). Ademas se pretende usar una base de datos con una amplia trayectoria en el mercado y casos de exito.
